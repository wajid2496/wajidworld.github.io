Linux Automation by using Jenkins

WE WILL BE PERMORMING BELOW STEPS TO MOVE OLD FILES FROM DIRECTORY /var/lib/Jenkins/workspace/Deployment/ TO /BACKUP BY JENKINS AUTOMATION.

1. It will find and list the old files, which are older than 30 Mints (we can do by Days also, It is just an example.)

2. It will send a mail notification and URL of job for Approval to Admin so Admin can check list of old files and approve in order to move files.

3. Once approved by admin, it will start moving old files to other directory or it will abort job if not approved in 1 Mint (We can define time according to our requirement.)

4. It will send mail notification also of job success or failed.
STEP 1: Click on new item >> Provide Name >> Select Pipeline Project>> Click ok

STEP 2: Mark Build periodically >> >> Click saveProvide cron job time of executiont[change time according to your need]

STEP 3: Paste your script >> Click save

STEP 4: Every 15 Mint Job will be build automatically.

STEP 5: STEP 5: Admin will receive mail notification and URL of Job.

STEP 6: STEP 6: After clicking on URL >> Console output


STEP 7: List of Old files in dir /var/lib/Jenkins/workspace/Deployment/

STEP 8: Click on input requested

STEP 9: Click proceed to move or Abort to cancel job.

STEP 10: Old Files will move to /backup/ directory if clicked on proceed.

STEP 11: Check mail notification of success job.
